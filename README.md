# MouseGestures
chrome 拡張。
画像のリンクをドラック＆ドロップ操作を行うことにより画像ファイルをダウンロード。

## インストール
[Chromeウェブストア](https://chrome.google.com/webstore/detail/draganddrop-gestures-mini/fenkeapgendkbamkllkaneehllpbobhg?hl=ja)
## 使い方。
画像のリンクをドラック＆ドロップ。
